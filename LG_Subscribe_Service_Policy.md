## 1. ประเภทบริการดูแลรักษา (Service Types)

### 🏠 VISIT — ช่างเข้าบริการถึงบ้าน

| หมวดสินค้า | รอบบริการ |
|---|---|
| เครื่องกรองน้ำ (WD516 / WD518 / WD110) | ทุก 6 เดือน |
| เครื่องกรองน้ำ Slim Stand (WS510) | ทุก 3 เดือน ⚠️ |
| ตู้เย็น Side-by-Side / Multi-Door (ทั่วไป) | ทุก 24 เดือน |
| ตู้เย็น Side-by-Side (Plumbing รุ่น X24/L24/X257CMHW) | ทุก 6 เดือน ⚠️ |
| ตู้เย็น Top/Bottom Freezer | ทุก 24 เดือน |
| Wash Tower | ทุก 12 เดือน |
| เครื่องซักผ้า Front Load | ทุก 12 เดือน |
| เครื่องซักผ้า Top Load | ทุก 12 เดือน |
| เครื่องอบผ้า / Styler | ทุก 12 เดือน |

| เครื่องฟอกอากาศ | ทุก 6 หรือ 12 เดือน (แล้วแต่แผน) |
| เครื่องลดความชื้น | ทุก 12 เดือน |

### 🛡️ No Service — รับประกันตลอดอายุสัญญา (ไม่มีบริการเข้าบ้าน)
ไม่มีช่างเข้าบริการประจำ แต่ได้รับประกันสินค้าตลอดอายุสัญญา แบ่งเป็น 2 กรณี:

#### กรณี A — ช่างเข้าซ่อมถึงบ้าน (On-site Repair)
เมื่อสินค้ามีปัญหา ติดต่อศูนย์บริการ LG → ช่างเดินทางมาซ่อมที่บ้านลูกค้า

| หมวดสินค้า |
|---|
| โทรทัศน์ (Life Style / OLED / QNED / NANO) |
| มอนิเตอร์ |

#### กรณี B — นำสินค้าส่งซ่อมที่ศูนย์บริการ LG (Drop-off Service)
เมื่อสินค้ามีปัญหา ลูกค้านำสินค้าไปส่งซ่อมที่ศูนย์บริการแอลจีด้วยตนเอง

| หมวดสินค้า |
|---|
| ไมโครเวฟ |
| ลำโพง / ซาวด์บาร์ (xboom, Sound Bar) |

## 3. ข้อยกเว้นรายรุ่น (Model-Specific Rules)

| รุ่น / กลุ่มรุ่น | ข้อยกเว้น | เหตุผล |
|---|---|---|
| **WS510SN** (Slim Stand PuriCare) | รอบบริการทุก **3 เดือน** | มีระบบกรองซับซ้อนกว่า (4 ชั้น) |
| **GC-X257CMHW**, **X24**, **L24** | ตู้เย็น Plumbing — ทุก **6 เดือน** + บริการต่อท่อน้ำ | มีระบบกรองน้ำและต่อท่อในตัว |

| **A9T-ULTRA** | SELF — จัดส่งอะไหล่ + Battery ครบ 36 เดือน | เป็นรุ่น high-end มีแบตสำรอง |

- สูตร: `ราคา/เดือน × 50% × 12 = ยอด Advance`
- ตัวอย่าง: สินค้าราคา 649 บาท/เดือน → 649 × 50% × 12 = **฿3,894**

### สินค้าที่ไม่มี Advance Payment

ไมโครเวฟ และสินค้าหมวดอื่นทั้งหมด

### 9.1 โครงสร้างไฟล์ PDF ของ LG

LG ออกแบบ Price List PDF แบบ **multi-column** แต่ละหน้ามีโครงสร้างดังนี้:

```
┌───────────────────────────────────────────────────────┐
│  [หัวหน้า] ชื่อโปรโมชัน  |  วันที่โปร (พ.ศ.)          │
├───────────────────────────────────────────────────────┤
│  [คอลัมน์ซ้าย]           │  [คอลัมน์ขวา]              │
│  Model Code              │  Model Code                 │
│  ชื่อสินค้า              │  ชื่อสินค้า                  │
│  5Y_Visit ทุกๆ 6 เดือน X│  5Y_Visit ทุกๆ 6 เดือน X  │
│  รอบบิลที่ 1 ถึง 6 (Y)  │  รอบบิลที่ 1 ถึง 6 (Y)    │
│  POLICY_CODE             │  POLICY_CODE                │
├───────────────────────────────────────────────────────┤
│  [ท้ายหน้า / Footer] ชื่อหมวดสินค้า  ←  สำคัญมาก!    │
└───────────────────────────────────────────────────────┘
```

**ข้อสำคัญ:** LG พิมพ์ชื่อหมวดสินค้าไว้ที่ **ท้ายหน้า (footer)** ไม่ใช่หัวหน้า  
`read_lg_pdf.py` จึงต้องตรวจทั้งส่วนต้น (8 บรรทัดแรก) และส่วนท้าย (10 บรรทัดสุดท้าย)

### 9.2 รูปแบบข้อมูลในแต่ละหมวด

#### รูปแบบ A — สินค้าทั่วไป (ตู้เย็น, แอร์, เครื่องซักผ้า ฯลฯ)

```
GC-X257CMHW.AEEPLMT
ตู้เย็น French Door with Craft Ice รุ่น X257CMHW
5Y_Visit ทุกๆ 6 เดือน 1,349
รอบบิลที่ 1 ถึง 6 (674)
รอบบิลที่ 7 ถึง 60 (1,349)
VISIT_5Y_6M_PRO_674(6M)
```

| ฟิลด์ PDF | ความหมาย |
|---|---|
| `5Y_Visit ทุกๆ 6 เดือน 1,349` | สัญญา 5 ปี, ช่างเข้าทุก 6 เดือน, ราคาปกติ 1,349 บาท/เดือน |
| `รอบบิลที่ 1 ถึง 6 (674)` | บิลที่ 1-6 จ่ายแค่ 674 (ช่วงโปรครึ่งราคา 6 เดือน) |
| `รอบบิลที่ 7 ถึง 60 (1,349)` | บิลที่ 7-60 ราคาปกติ |
| `VISIT_5Y_6M_PRO_674(6M)` | Policy Code สำหรับ LG Internal |

#### รูปแบบ B — TV / Monitor / ลำโพง (No Service + Advance Payment)

```
55QNED80BSA.ATM
LG 55" QNED 80 4K
3,894  649
รอบบิลที่ 1 ถึง 6 (324)
รอบบิลที่ 7 ถึง 60 (649)
NOSERVICE_5Y_PRO_ADV50%(6M)_DC50%(6M)
```

| ฟิลด์ PDF | ความหมาย |
|---|---|
| `3,894  649` | Advance Payment 3,894 บาท (จ่ายล่วงหน้า), ราคารายเดือน 649 บาท |
| `รอบบิลที่ 1 ถึง 6 (324)` | ช่วงโปร: เดือนที่ 1-6 จ่าย 324 บาท |
| `NOSERVICE_5Y_...` | Policy Code รูปแบบ No Service |

> ⚠️ **Dot เป็น Thousands Separator** — PDF บางหน้า OCR ผลิต `1.149` แทน `1,149`  
> Script แก้ด้วยการลบทั้ง `,` และ `.` แล้ว parse เป็นตัวเลขล้วน

### 9.6 ข้อจำกัดและจุดระวัง

| ปัญหา | สาเหตุ | วิธีรับมือ |
|---|---|---|
| ลำโพง xboom ไม่มี Model Code ชัดเจน | PDF พิมพ์แค่ชื่อรุ่นภาษาไทย ไม่มีรหัสแบบ alphanumeric | ต้องอัปเดต KNOWN_PREFIXES หรือ looks_like_model() ด้วยตนเอง |
| Multi-column PDF ทำให้ข้อมูลปนกัน | pypdf linearize หน้า → คอลัมน์ซ้ายและขวาปนกัน | Script ใช้ pattern matching แทน position-based parsing |
| ราคา dot แทน comma (`1.149`) | OCR artifact ใน PDF | Strip ทั้ง `,` และ `.` ก่อน parse |
| หมวดสินค้าไม่ตรงหน้า | LG พิมพ์ชื่อหมวดที่ footer ไม่ใช่ header | `detect_category()` ตรวจ 8 บรรทัดแรก + 10 บรรทัดท้าย |
| TV model มีตัวเลขนำหน้า (`55QNED...`) | `looks_like_model()` เดิม reject digit-start | `RE_TV_MODEL` ตรวจก่อน digit-rejection rule |
| LG ออก V2 PDF กลางเดือน (ราคาเปลี่ยน) | LG แก้ไข PDF หลังออก V1 | รัน `--diff` ระหว่าง V1 กับ V2 เพื่อดูว่าเปลี่ยนอะไร |

### 9.7 ตัวแปรสำคัญใน Script

| ตัวแปร | ที่อยู่ | ใช้ทำอะไร |
|---|---|---|
| `PAGE_CAT` | บรรทัด 36-67 | map keyword ใน footer → ชื่อหมวดสินค้า |
| `KNOWN_PREFIXES` | บรรทัด 99-112 | prefix ที่รู้จัก สำหรับ detect Model Code |
| `EXCLUDE_FIRST` | บรรทัด 113-118 | คำที่ขึ้นต้นบรรทัดแล้ว **ไม่ใช่** model code |
| `NO_SERVICE_CATS` | บรรทัด 240-244 | หมวดที่ใช้รูปแบบ Advance+Monthly แทน RE_CONTRACT |
| `RE_TV_MODEL` | บรรทัด 121 | regex สำหรับรหัส TV/Monitor ที่เริ่มด้วยตัวเลข |

### 9.8 เมื่อ PDF เดือนใหม่มีหมวดสินค้าใหม่

ถ้า Script แสดงรุ่นน้อยกว่าปกติ หรือบางหมวดหายไป:

1. รัน script แล้วดูหมวดไหนหาย
2. เปิด PDF ดูหน้านั้น → ดู footer ว่าพิมพ์ชื่ออะไร
3. เพิ่ม keyword ใน `PAGE_CAT` ใน `read_lg_pdf.py`
4. ถ้าเป็นหมวด No Service ใหม่ → เพิ่มใน `NO_SERVICE_CATS`
5. ถ้า Model Code รูปแบบใหม่ → เพิ่มใน `KNOWN_PREFIXES` หรือแก้ `RE_TV_MODEL`
